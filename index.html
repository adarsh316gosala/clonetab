Welcome to The clonetab



Wel Come To CloneTab

Clonetab is a software appliance for cloning automation & virtualization of Oracle e-Business Suite, PeopleSoft and Databases such as Oracle, MySQL, Hana, SQL Server and so on. The company was created with the ambition of providing an innovative product to companies to increase profitability and productivity while minimizing the time and money spent on manual, monotonous cloning of software. In essence, Clonetab serves the role of multiple employees, making it more efficient to companies who can save on unnecessary costs. Built on the Java Platform, Clonetab’s secure cloning software offers advantageous speedy solutions to all its customers. Among other things, Database Administrators (DBAs), spend considerable amount of time on tasks such as refreshes, patching, and start and stop operations of various instances, which tend to be complex, tedious processes. Should potential businesses choose to make use of this software, it will allow for them to utilize their DBAs for other valuable tasks. Through Clonetab, customers are entitled to many benefits which include, but are not limited to: customizations to the refresh procedures to automate and tune them; 24/7 Remote DBA Services; and installation, configuration, implementation, and support services for Clonetab. Other valuable features include a Netapp storage module that allows for more economical and quicker refreshes and other related tasks; a user-friendly graphical user interface (GUI), role-based access to DBA and Unix System administrations; and a password wallet to safeguard confidential information. Several fortune 500 companies across the globe use Clonetab products in the areas of ERP cloning/Virtualization, Database cloning/virtualization, DR, Data Scrambling, SQL Self Service, ERP Backup, Database Backup and so on.
